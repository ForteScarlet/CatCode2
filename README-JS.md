## 安装

### npm

> npm package see: https://www.npmjs.com/package/@catcode2/core

```shell
npm i @catcode2/core
```

## 使用

### import

```javascript
import {catcode2 /* and other... */} from '@catcode2/core' 
```

### require

```javascript
const catcode2 = require('@catcode2/core').catcode2
```
